# Matrix and vector prerequisite

## Dot product

A dot product combines two vectors into one number by multiplying corresponding components and adding the results. In attention, the dot product is used as a similarity-related score between a query and a key.

## Weighted sum

A weighted sum multiplies each value by a weight and adds the results. Larger weights make the corresponding values contribute more to the new representation.

## Why this matters for attention

Attention scores become weights. Those weights determine how much information from each value is included in the output for the current position.
