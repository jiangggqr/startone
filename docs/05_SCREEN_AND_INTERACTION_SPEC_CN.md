# 05 逐屏 UI 与交互规范

每个页面都必须实现目标、主操作、次操作、数据来源、状态和验收。编号与原型一致。

## 00 总览

仅作为设计文档，不进入生产导航。说明完整闭环、系统边界和实现顺序。

## 01 首页与资料库

- 唯一主操作：上传材料并新建学习会话
- 次操作：继续最近会话
- 上传材料入口强于“仅输入主题”
- 显示最近会话、进度和下一动作
- 空状态包含添加第一份材料的按钮

## 02 目标与偏好

字段：目标、基础、时间、精力、语言、当前问题、解释偏好、时间显示、搜索建议权限。

- 必填错误与字段关联
- 选择应自动保存
- 搜索权限说明“每次搜索前仍确认”
- 不要求用户填写复杂 Prompt

## 03 上传与解析

- 每个文件独立状态：等待、上传、解析、成功、部分成功、失败、取消
- 显示格式和大小限制
- 无文字扫描 PDF 明确提示
- 用户可移除失败文件并继续成功文件
- 来源结构和片段可点开定位
- 上传前显示具体格式、单文件大小、文件数限制
- 处理中显示当前阶段和进度；超过 10 秒提供取消或后台继续

## 04 来源覆盖

- 区分已覆盖与候选缺口
- 明确尚未联网
- 缺口不自动等于搜索任务
- 显示三类来源标签

## 05 学习地图

- 显示概念依赖和本次路线
- 显示预计时间和概念数
- 允许调整重点、减少概念或改变时间
- 第一个动作必须有时长和完成条件

## 06 两分钟启动

- 页面只显示一个任务
- 允许暂停
- 错误不扣分
- 用户输入自动保存
- 完成后进入第一个概念

## 07 专注工作区

- 左：紧凑地图
- 中：当前概念、作用、简练解释、来源、一个主动作
- 右：时间、进度、保存、退出
- 主动作视觉唯一最强
- Tutor、Quiz、复述和来源为次操作
- 时间可隐藏

## 08 Tutor

- 显示当前概念和来源类型
- 快捷问题与自由输入同时存在
- 首选连续引导，不默认直接给完整答案
- 对话可恢复
- 超出范围时标记前置知识或添加待办，不伪造来源

## 09 Quiz

- 一次一题
- 干扰项对应误解标签
- 正确答案提交前不可见
- 提示逐级显示
- 提交后显示为什么，不只显示对错

## 10 自由复述

- 任务要求清楚且短
- 可用 0–3 级提示
- 未提交输入刷新后保留
- 允许返回讲解

## 11 即时反馈

固定结构：

1. 已掌握
2. 需要补充或澄清
3. 简练修正答案
4. 具体鼓励
5. 推荐下一微动作

反馈避免人格评价和羞辱语言。

## 12 补救练习

- 由最新具体误区驱动
- 支持更简单解释、更小问题、具体例子
- 连续失败后切换策略
- 完成后形成证据并进入 Agent 决策

## 13 Agent 决策

- 左侧显示证据摘要
- 右侧只高亮一个推荐动作
- 显示理由和预计时间
- 用户可选择其他路径
- 其他路径只列出当前状态允许的动作；不可用动作不伪装成可选项
- Agent 不在该页生成新的教学内容

## 14 搜索确认

- 显示命名缺口
- 显示要搜索什么和为何搜索
- 用户可拒绝且不中断学习
- 搜索尚未发生时不得写“已找到”

## 15 外部来源选择

- 默认展示少量候选
- 每项显示标题、URL、可信类型、选用原因和填补缺口
- 用户可预览、使用、忽略或重新搜索
- 外部来源必须持续标注
- 每项必须显示规范 URL、发布者、访问时间、引用片段和选择原因

## 16 来源查看器

- PDF 显示页码；Markdown/TXT 显示标题和行号
- 标出该片段支持的具体内容
- 支持报告定位错误
- 关闭后返回原触发位置
- 默认 Demo 使用 Markdown 标题与行号；独立 PDF fixture 才显示真实页码

## 17 总结与恢复

- 已完成概念
- 仍需复习
- 主要误区和已解决问题
- 下次 1–2 分钟启动动作
- 保存退出、继续和导出

## 18 学习记录

- 继续、查看总结、复制、删除
- 删除前明确范围
- 搜索和筛选会话

## 19 设置与隐私

- 学习体验、无障碍、搜索权限、数据下载与删除
- AI 活动和搜索记录可查看
- 破坏性操作二次确认

## 20 系统状态

以下状态必须各有可见示例和清楚恢复动作：

- 加载：上传中、解析中、地图生成中、Tutor streaming、活动提交中、搜索中
- 空：无材料、无历史会话、无搜索结果、无可用替代路径
- 部分成功：多文件部分解析、多个模型子任务部分成功
- 错误：格式/大小错误、解析失败、来源引用无效、模型输出无效、Tutor/活动/API 失败、搜索失败、权限拒绝、删除失败
- 取消与重试：用户取消、后台继续、重试中、重试后成功/仍失败
- 保存：正在保存、已保存、离线保存、保存失败、版本冲突与版本选择、重新同步成功
- 恢复：刷新恢复、暂停恢复、离线恢复、搜索失败返回原状态

## 21 响应式与键盘

移动端完整支持核心闭环，不能只提供一张缩小示意图；键盘顺序、页面切换后的标题焦点、抽屉/模态焦点返回、Tab/Radio/Disclosure 语义控件必须验证。
