## Description

StartOne turns uploaded learning material into a visual knowledge map, then guides the learner through one focused concept at a time: explanation, contextual Tutor support, active retrieval, immediate feedback, and one safe next step.

Built for the OpenAI Build Week Education track with Codex, GPT-5.6, the OpenAI Responses API, FastAPI, SQLite, and dependency-free HTML/CSS/JavaScript.

Try StartOne:
https://startone-learning.onrender.com

Source code:
https://github.com/jiangggqr/startone

No account or personal API key is required. Uploaded material is the only learning source. If information is missing, StartOne asks the learner to add the relevant material instead of searching the web.

This demo includes English narration and burned-in English captions. The voice is AI-generated.
